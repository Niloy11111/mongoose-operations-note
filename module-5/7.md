#### the documents where skills is an array and each object in that array has name property value as JAVASCRIPT

```javascript
db.test
  .find({
    "skills.name": "JAVASCRIPT",
  })
  .project({ friends: 1 });
```

#### the documents where skills is an array and level is Intermidiate

```javascript
db.test
  .find({
    skills: {
      $elemMatch: {
        name: "JAVASCRIPT",
        level: "Intermidiate",
      },
    },
  })
  .project({ skills: 1 });
```
